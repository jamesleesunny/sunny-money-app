# Sunny Money PWA

이 폴더를 정적 웹호스팅에 올리면 아이폰 Safari에서 `홈 화면에 추가`가 가능한 웹앱입니다.

포함 파일:
- index.html
- manifest.webmanifest
- sw.js
- icons/

## 가장 쉬운 배포 방법

### GitHub Pages
1. GitHub에서 새 저장소 생성
2. 이 폴더 파일 전체 업로드
3. Settings → Pages → Deploy from a branch → main / root 선택
4. 생성된 주소를 아이폰 Safari에서 열기
5. 공유 버튼 → 홈 화면에 추가

### Netlify Drop
1. Netlify Drop 페이지 열기
2. 이 폴더 전체 또는 zip 업로드
3. 생성된 주소를 아이폰 Safari에서 열기
4. 공유 버튼 → 홈 화면에 추가

## 주의
- 아이폰 파일 앱에서 로컬 HTML로 열면 `홈 화면에 추가`가 안 보일 수 있습니다.
- 반드시 `https://` 웹주소로 열어야 합니다.
