아이폰에서 앱처럼 설치하기

1. 이 폴더를 GitHub Pages, Netlify, Vercel 중 하나에 업로드합니다.
2. 아이폰 Safari에서 배포 주소를 엽니다.
3. Safari 공유 버튼 → 홈 화면에 추가를 누릅니다.
